Problem Statement: Create a JavaScript program that takes user input and displays output using conditions and loops. Create a web application that: 1. Takes the number of subjects from the user. 2. Uses a loop to take marks for each subject. 3. Calculates: o Total Marks o Average Marks 4. Uses if-else conditions to display: o Grade o Pass/Fail Result 5. Displays the result dynamically on the webpage. Objective • To understand JavaScript variables and data types • To implement loops for repetitive tasks • To apply conditional statements (if–else) • To dynamically display results using DOM manipulation Tools Required: HTML, JavaScript, CSS Theory JavaScript is a client-side scripting language used to create dynamic web pages. Important concepts used in this experiment: 1. Variables Variables are used to store data values. Example: let total = 0; 2. Loops Loops are used to execute a block of code repeatedly. Example: for (let i = 1; i <= count; i++) 3. Conditional Statements Conditional statements execute different code based on conditions. Example: if (average >= 75) 4.
